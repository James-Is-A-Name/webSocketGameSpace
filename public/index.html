Stick figure paper scissors rock

This is a simple game that uses multiple browser to allow multiple people to participate with it.

The "game" consists of stick figures of three styles moving around competeing when colliding. Green is shield, Blue is bow, Red is sword. Green beats Blue, Blue beats Red, Red beats Green.

A browser acting as a display will be where players run around. Connecting display instances will allow players to travel between them. Connecting as a controller will create a player that it controls on a display.

The way the connections are accomplished without extreme delay is with the use of webRTC. In its current state no TURN/STUN servers are used so it will only work when all devices intending to connect are on the same local network.
